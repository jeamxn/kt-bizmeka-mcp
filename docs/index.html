설치 방법
  1. 위에서 본인 OS에 맞는 .zip을 받아 압축을 풉니다.
  2. macOS / Linux는 실행 권한이 필요할 수 있습니다. 
    chmod +x kt-bizmeka-mcp
  3. Claude Code에 로컬 플러그인으로 추가합니다. 압축 안의 .claude-plugin/plugin.json이 바로 그 바이너리를 실행하도록 구성돼 있습니다.
  4. Claude에서 bizmeka_man을 먼저 호출해 사용 가능한 도구와 로그인 흐름을 확인한 뒤 시작하세요.

macOS에서 “확인되지 않은 개발자” 경고가 뜨면 xattr -d com.apple.quarantine kt-bizmeka-mcp 로 해제할 수 있습니다.
